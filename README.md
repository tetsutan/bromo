
# Bromo : Broadcast monitor

![](/docs/img/logo.png)

音声・動画データを取得してposdcast形式で配信するサーバーです
単品の録音データを取得するのではなく、スケジュールリストから自動で録音・録画を行います。
例えば、一階限りの
